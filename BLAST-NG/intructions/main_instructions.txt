Instructions:

1. Click the "Add Database" button to build a database
     
2.Input the sequence to be aligned into the text box or select the sequence file through the "Select file" button      

3.Select the database to be compared and the comparison method

4.Set the e-value Value, output format and number of threads.The default e-value =1e-5, and the default output format is 0 and the default of threads is 4
      
5.(Optional) Enter additional BLAST commands such as: -max_target_seqs 20 

6.Click the "Start" button for comparison. Results will be displayed in the output window as well as saved to a file named result.txt

Note: Alignment time depends on sequence size and computer performance
