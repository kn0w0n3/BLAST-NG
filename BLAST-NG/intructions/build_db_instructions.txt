1. Click the select file button to select a FASTA file. 

2. Select a database type (Nucleic Acid Sequence or Protein Sequence).
            
3. Enter a name for the database (no Spaces).
            
4. Click the Build database button to start building the database Databases will be stored in C:\BLAST-NG\databases.
            
5. After the database build process has completed, it will be listed in the database dropdown on the main window.

Databases can be downloaded from the following link: https://ftp.ncbi.nlm.nih.gov/blast/db/FASTA/
